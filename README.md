## App-Demo-Web
不推荐全局安装 webpack。这会将你项目中的 webpack 锁定到指定版本，并且在使用不同的 webpack 版本的项目中，可能会导致构建失败。


## Npm
```shell
# create package.json
npm init

# install
npm install

# build
npm run build
```


## Webpack
```shell
# webpack
npm install --save-dev webpack webpack-cli

# lodash
npm install --save lodash

# pack
npx webpack
```


## Gulp
```shell
# gulp-cli
npm install --save-dev gulp-cli

# cd my-project
npm init
npm install --save-dev gulp
```
